HTML {} Este seletor faz referência à tag root ou raiz em uma página web

  • html {
  • margin: 0px;
  • padding: 0px;
  • }

HEAD {} Seletor cabeçalho é uma tag que fornece informações gerais ou metadados sobre o documento.

| Sem aplicações práticas em css

Title {} Seletor usado na tag head para definir na aba do navegador um título.

| Sem aplicações práticas em css

Meta {} Seletor usado na tag head para definir na aba do navegador um título.

| Sem aplicações práticas em css

Body {} Seletor especifica as relações entre o documento atual e um recurso externo

  • body {
  • color: grey;
  • background-color: #fce;
  • }

Link {} Seletor especifica as relações entre o documento atual e um recurso externo

  • link {
  • color: red;
  • padding: none;
  • }

h1 {} Seletor representa um título varia de h1 ate h6, sendo h2 o subtitulo de h1

  • h1 {
  • text-align: center;
  • font-size: 3em;
  • }

p {} Seletor para texto qualquer a letra "p" é uma abreviação de parágrafo, mas e usado para estruturar seus textos

  • p {
  • text-align: left;
  • color: blue;
  • }
